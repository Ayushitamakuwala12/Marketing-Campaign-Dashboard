# Marketing Campaign Performance Dashboard

# Project Overview
This project is an interactive Excel/Power BI dashboard developed to analyze marketing campaign performance and business KPIs. The dashboard helps understand revenue trends, campaign effectiveness, conversion efficiency, and channel-wise performance.

# Objectives
- Analyze overall marketing performance
- Track campaign revenue and ROI
- Monitor conversion funnel
- Compare channel effectiveness
- Generate business insights for decision-making

# Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Power Query
- DAX (Power BI)
- Data Visualization

# Key KPIs
- Total Revenue
- Total Cost
- ROI
- Conversion Rate
- Revenue Per Click
- Total Campaigns
- Total Conversions

# Dashboard Pages

# 1. Overview Dashboard
- Revenue trend analysis
- ROI by channel
- Marketing conversion funnel
- Cost vs revenue analysis

# 2. Campaign Analysis
- Top performing campaigns
- Campaign performance matrix
- Conversion efficiency
- ROI analysis

# 3. Channel Analysis
- Revenue share by channel
- Conversion rate comparison
- Channel performance matrix
- Cost vs revenue by channel

# 4. Insights & Recommendations
- Business findings
- Performance insights
- Strategic recommendations

# Business Insights
- Search campaigns generated highest overall revenue.
- Influencer campaigns showed high spending with comparatively lower ROI.
- Email campaigns maintained strong conversion efficiency at lower cost.
- Conversion funnel analysis highlighted customer drop-offs between clicks and conversions.
